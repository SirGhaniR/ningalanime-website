# 🎬 NINGAL ANIME

Selamat datang di **NINGAL ANIME** — sebuah platform website untuk menikmati berbagai anime favorit dengan pengalaman yang simpel, cepat, dan modern.

---

## 🚀 Tentang Project

**NINGAL ANIME** adalah website streaming anime yang dirancang untuk memberikan kemudahan kepada pengguna dalam menonton anime kapan saja dan di mana saja. Dengan tampilan yang clean dan fitur yang lengkap, website ini cocok untuk para pecinta anime.

---

## 🛠️ Teknologi yang Digunakan

Website ini dibangun menggunakan teknologi modern:

* **Backend** : Express JS
* **Frontend** : React JS
* **Database** : MongoDB

---

## ✨ Fitur Utama

Berikut beberapa fitur yang tersedia di dalam website ini:

* 🏠 **Home Page** — Halaman utama berisi pengenalan website
* 🔐 **Login & Register** — Sistem autentikasi user
* 📄 **Login/Register Page** — Halaman khusus untuk masuk & daftar
* 📊 **Dashboard** — Panel admin untuk mengelola konten
* 🎥 **Nonton Anime** — Fitur streaming anime
* 🕓 **History** — Menyimpan riwayat tontonan user
* 🧩 **CRUD System** — Admin dapat menambah, mengedit, dan menghapus anime
* 🧭 **Navbar** — Navigasi yang user-friendly
* 📌 **Footer** — Informasi tambahan website

---

## 🔄 Alur Sistem (Flow)

1. Saat pertama kali membuka website, user akan diarahkan ke **Home Page**
2. Home Page berisi informasi dan pengenalan tentang **NINGAL ANIME**
3. Jika user ingin menonton anime, maka diwajibkan untuk **login atau register terlebih dahulu**
4. Admin dapat menambahkan anime melalui **Dashboard**
5. User yang sudah login dapat menonton anime dan melihat **history tontonan**

## ⚙️ Instalasi & Setup

```bash
# Clone repository
git clone https://github.com/username/ningal-anime.git

# Masuk ke folder project
cd ningal-anime

# Install dependencies backend
cd backend
npm install

# Install dependencies frontend
cd ../frontend
npm install

# Jalankan backend
npm run dev

# Jalankan frontend
npm start
```

---

## 📌 Catatan

* Pastikan MongoDB sudah berjalan sebelum menjalankan server
* Gunakan file `.env` untuk menyimpan konfigurasi seperti database dan JWT
