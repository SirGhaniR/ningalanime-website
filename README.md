PERANCANGAN WEBSITE & TOOLS YANG DIGUNAKAN

BACKEND : EXPRESS JS
FRONTEND: REACT JS
DATABASE: MONGO DB

NINGAL ANIME, "NINGAL ANIME" ini adalah website yang bertujuan untuk menonton berbagai anime

A. Fiturnya: 
- Dashboard 
- Login/Register
- Login/Register Page
- History
- Nonton
- CRUD
- Home Page
- Navbar
- Footer

B. Flow Chart: 
1. Ketika user baru pertama kali masuk, maka akan otomatis kedalam Home Page.
2. Home Page berisikan tentang pengenalan Website "NINGAL ANIME"
3. Ketika user ingin menonton anime maka diharuskan untuk login/register terlebih dahulu.
4. Anime dapat ditambahkan oleh Admin sendiri di Dashboard.
